# 주식 시장 대시보드

실시간 주식 정보를 보여주는 웹 애플리케이션입니다.

## 기능

- 실시간 주식 가격 및 변동률 표시
- 한 달간 최고 상승주 표시
- 인터랙티브 그래프 (더블클릭으로 상세 정보)
- matplotlib을 사용한 고품질 차트 다운로드

## 설치 및 실행

### 1. Python 백엔드 설정

```bash
# 필요한 패키지 설치
pip install -r requirements.txt

# 서버 실행
python app.py
```

서버는 `http://localhost:5000`에서 실행됩니다.

### 2. 웹 애플리케이션 실행

`index.html` 파일을 브라우저에서 열면 됩니다.

## 사용 방법

1. **주식 정보 보기**: 각 주식 카드에서 실시간 가격과 변동률을 확인할 수 있습니다.
2. **그래프 상세 보기**: 그래프를 더블클릭하면 상세 정보 모달이 열립니다.
3. **차트 다운로드**: 모달에서 "matplotlib 차트 다운로드" 버튼을 클릭하면 고품질 PNG 이미지로 다운로드됩니다.
4. **네이버 주식 페이지 이동**: 주식 카드를 더블클릭하면 네이버 주식 페이지로 이동합니다.

## 필요한 패키지

- Flask: 웹 서버
- Flask-CORS: CORS 처리
- matplotlib: 그래프 생성
- requests: API 요청

## 주의사항

- Python 서버가 실행 중이어야 차트 다운로드 기능을 사용할 수 있습니다.
- 주식 데이터는 Yahoo Finance API를 통해 가져옵니다.
